
【アクアボトルキャッチャー 熱血サトウくん】使い方

このゲームを動かすには、以下のファイルをすべて同じフォルダに入れてください：

1. aqua_bottle_game_with_file_bgm.html（このファイル）
2. BGM.mp3（BGM音楽ファイル）
3. bakuhatu.mp3（爆発音ファイル）
4. sato.png（キャラクター画像）
5. bottle.png（普通のボトル画像）
6. red_bottle.png（赤いボトル画像）

【起動方法】
・aqua_bottle_game_with_file_bgm.html をブラウザでダブルクリックするだけ！

【注意】
・ファイル名は正確に！（例：BGM.mp3 は BGM.MP3 や bgm.mp3 では動きません）
・音が出ない場合は、ブラウザの設定（音量や自動再生制限）も確認してみてください。

がんばれサトウくん！
